# 서비스 비전 - AI 인생설계 플랫폼

## 서비스 정의

### 한 줄 정의
> 사주/점성술 분석 + AI 코칭 + 게이미피케이션을 결합한 **인생설계 플랫폼**

### 핵심 철학
단순히 "오늘 운세가 좋습니다"로 끝나는 것이 아닌,  
**"그래서 오늘 무엇을 하면 좋을지"**까지 안내하는 서비스

---

## 왜 이 서비스인가?

### 기존 운세앱의 한계

점신, 포스텔러 등 기존 앱들은 각자의 강점이 있지만, 공통적으로:
- 운세 결과를 보여주고 **끝**
- "좋은 날이에요" → 그래서 뭘 하라는 건지?
- 매일 보지만 **실제 행동 변화로 연결되지 않음**

### 우리의 관점

운세는 **방향을 알려주는 나침반**이지, 목적지가 아닙니다.

```
기존: 운세 확인 → 끝

우리: 운세 확인 → 오늘의 액션 제안 → 실행 → 피드백 → 성장
```

## 핵심 기능 설계

### 1. 다중 성향 분석 통합

| 분석 체계 | 기반 | 역할 |
|-----------|------|------|
| **사주팔자** | 동양 명리학 | 타고난 기질, 대운/세운 타이밍 |
| **점성술** | 서양 천문학 | 성격, 관계 스타일 |
| **MBTI/에니어그램** | 심리학 | 행동 패턴, 소통 스타일 |

**벤치마킹 포인트:**
- 점신의 **깊이 있는 사주 분석** 수준 유지
- 포스텔러의 **스토리텔링** 방식 적용
- 여러 체계를 **교차 분석**하여 더 입체적인 인사이트

### 2. 인생 설계 시스템

Fabulous가 증명한 "목표 설정 → 습관 형성" 시스템을  
**사주의 대운/세운 개념**과 결합합니다.

```
🌟 인생 로드맵 (대운 기반)
│
├── 📆 10년 설계
│   "이 대운 기간의 핵심 테마는?"
│   └── 세운(歲運) 기반 연도별 흐름
│
├── 📅 1년 설계
│   "올해의 핵심 목표 3가지"
│   └── 월운(月運) 기반 분기별 전략
│
├── 📋 1달 설계
│   "이번 달 집중할 것"
│   └── 주간 마일스톤
│
└── ✅ 하루 설계
    "오늘의 3가지 액션"
    └── 일진(日辰) 기반 시간대별 가이드
```

### 3. 실시간 Google Grounding

**Co-Star**는 위치 기반 별자리 정보를 제공합니다.  
우리는 여기서 한 발 더 나아갑니다.

| 기존 운세 | 우리 서비스 (Grounding) |
|----------|----------------------|
| "금전운이 좋습니다" | "오늘 코스피 2% 상승 중, 당신의 재성과 맞아요" |
| "건강 주의하세요" | "오늘 서울 미세먼지 나쁨, 실내 운동 추천" |
| "여행운이 좋아요" | "제주 겨울빛 축제 이번 주말, 여행 적기" |

**실시간 뉴스, 날씨, 이벤트 + 개인 사주 = 맥락화된 조언**

### 4. AI 캐릭터 코칭

**헬로봇**이 증명한 캐릭터의 힘을 활용합니다.

#### 오행 기반 코치 캐릭터

| 오행 | 캐릭터 | 성격 | 코칭 스타일 |
|-----|--------|-----|-----------|
| 木 | 목신 | 성장형, 따뜻함 | 격려, 잠재력 발견 |
| 火 | 화신 | 열정형, 직설적 | 동기부여, 도전 유도 |
| 土 | 토신 | 안정형, 신중함 | 체계적 계획, 안정감 |
| 金 | 금신 | 결단형, 날카로움 | 현실적 조언, 결단 |
| 水 | 수신 | 유연형, 지혜로움 | 통찰, 감정 공감 |

**사용자 사주의 용신(用神)에 맞는 코치 자동 매칭**

### 5. 게이미피케이션

**Duolingo**의 스트릭, **헬로봇**의 캐릭터 성장을 참고합니다.

- **일일 퀘스트**: 오늘 운세에 맞는 미션 3가지
- **스트릭 시스템**: 연속 접속 보상
- **오행 밸런스 게이지**: 목화토금수 에너지 균형 시각화
- **캐릭터 성장**: 코치 캐릭터가 함께 레벨업

### 주요 세그먼트

| 세그먼트 | 니즈 | 우리가 제공하는 가치 |
|----------|------|-------------------|
| **20대 취준생** | 진로 결정 | 적성 분석 + 구체적 액션플랜 |

| **예비 부부** | 결혼 준비 | 궁합 + 함께하는 인생 계획 |
| **창업 준비자** | 사업 시작 | 사업운 + 파트너 궁합 + 실행 계획 |

### 차별화 포인트

기존 운세앱은 **"오늘 어때요?"**만 알려줍니다.  
우리는 **"그래서 오늘 이걸 하세요"**까지 제안합니다.

## 일일 사용자 경험

### 아침 (7-9시)
```
🌅 좋은 아침이에요!

오늘의 에너지: 🔥🔥🔥⚪⚪
주요 기운: 화(火) 기운 상승

오늘의 3가지 추천:
1. 오전 10시 전에 중요한 결정하기 ⭐
2. 점심은 밖에서 (날씨 맑음 + 화 기운)
3. 저녁 운동 30분 (목 기운 보충)

[상세 운세 보기] [퀘스트 시작]
```

### 저녁 (21-23시)
```
🌙 오늘 하루 어땠나요?

완료한 퀘스트: 2/3 ✅
획득한 경험치: 150 XP
연속 접속: 7일 🔥

내일 미리보기:
금(金) 기운 상승 → 재테크 검토 적기

[내일 퀘스트 미리보기] [수면 모드]
```
