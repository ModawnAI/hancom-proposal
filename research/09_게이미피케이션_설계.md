# 게이미피케이션 설계

## 핵심 메커니즘

### 1. 스트릭 시스템

일일 체크인 연속 기록

```
🔥 현재 스트릭: 127일

[일] [월] [화] [수] [목] [금] [토]
 ✓    ✓    ✓    ✓    ✓    ✓    ?

⚠️ 오늘 체크인하지 않으면 스트릭이 끊어집니다!
```

**보상 체계:**
| 스트릭 | 보상 |
|--------|------|
| 7일 | 🎁 레어 아이템 상자 |
| 30일 | 🎁 에픽 아이템 + 캐릭터 스킨 |
| 100일 | 🎁 레전더리 + 특별 칭호 |
| 365일 | 🎁 연간 회고 + 프리미엄 보상 |

**심리적 효과:** 손실 회피 (끊기 싫음)

---

### 2. 인생 RPG 시스템

```
나의 인생 스탯

[커리어]  ████████░░░░ 67/100
[건강]    ██████░░░░░░ 52/100
[재정]    █████░░░░░░░ 45/100
[관계]    ███████████░ 89/100
[성장]    ████████░░░░ 71/100

━━━━━━━━━━━━━━━━━━━━
📜 진행 중인 퀘스트:
• [메인] 스타트업 창업 (17% / 보상: 커리어 +30)
• [서브] 체중 5kg 감량 (60% / 보상: 건강 +15)
• [일일] 오늘의 미션 3개 (1/3)

🏆 최근 업적: "첫 비즈니스 계획서 작성" 🎉
```

**스탯 영역:** 오행(五行)과 연계
- 커리어 = 木 (성장, 발전)
- 건강 = 水 (생명력)
- 재정 = 金 (재물)
- 관계 = 火 (열정, 교류)
- 성장 = 土 (안정, 기반)

### 3. 레벨 시스템

```
인생 레벨 = f(스탯 총합, 퀘스트 완료, 스트릭)

레벨 1-10: 초보 탐험가
레벨 11-30: 방향 탐색자
레벨 31-50: 목표 추구자
레벨 51-70: 인생 설계사
레벨 71-99: 마스터
레벨 100: 현자 (히든)
```

**XP 획득:**
| 활동 | XP |
|------|-----|
| 일일 체크인 | +10 |
| 일일 미션 완료 | +15/개 |
| 주간 목표 달성 | +50 |
| 월간 목표 달성 | +200 |
| 퀘스트 완료 | +100~500 |

### 4. 캐릭터 시스템

#### 12지신 캐릭터
- 사용자 띠에 따라 수호 캐릭터 배정
- 캐릭터가 운세 전달 + 코칭 제공

| 띠 | 캐릭터 | 성격 |
|----|--------|------|
| 쥐 | 지혜의 쥐 | 영리함, 분석적 |
| 소 | 우직한 소 | 꾸준함, 인내 |
| 호랑이 | 용맹한 호랑이 | 용기, 도전 |
| 토끼 | 온화한 토끼 | 부드러움, 배려 |
| 용 | 청룡 | 카리스마, 리더십 |
| 뱀 | 현명한 뱀 | 직관, 지혜 |
| 말 | 자유로운 말 | 열정, 활력 |
| 양 | 평화로운 양 | 조화, 예술성 |
| 원숭이 | 재치있는 원숭이 | 창의력, 유머 |
| 닭 | 근면한 닭 | 성실함, 정확함 |
| 개 | 충직한 개 | 충성, 정의감 |
| 돼지 | 복스러운 돼지 | 풍요, 관대함 |

#### 캐릭터 성장
- 사용할수록 호감도 증가
- 레벨업 시 새로운 스킨/능력 해금
- 캐릭터 진화 시스템

---

### 5. 컬렉션 시스템

#### 캐릭터 컬렉션
- 12지신 기본 캐릭터
- 시즌/이벤트 한정 스킨
- 레벨업 진화형

#### 테마 컬렉션
- 사주 테마 (오행, 십신, 십이운성)
- 계절/절기 테마
- 업적 테마

#### 업적 배지
| 업적 | 조건 |
|------|------|
| 첫 발걸음 | 첫 목표 설정 |
| 불꽃 수호자 | 100일 스트릭 |
| 인생 설계사 | 레벨 50 달성 |
| 궁합 마스터 | 친구 10명 궁합 확인 |
| 계획의 달인 | 월간 목표 12회 달성 |

---

### 6. 소셜 기능

#### 궁합 보기
- 친구/연인/동료와 사주 궁합
- "이 사람과 함께할 때 좋은 활동"
- 공유용 궁합 카드 생성

#### 목표 공유
- 선택적으로 목표 공개
- 친구가 응원 보내기
- 함께하는 챌린지

#### 리그 시스템
```
💎 다이아몬드
     ↑
🏆 골드
     ↑
🥈 실버   ← 현재 위치 (상위 15%로 승급)
     ↑
🥉 브론즈
```

- 주간 리셋으로 신선함 유지
- 상위 N% 승급, 하위 N% 강등
- 같은 띠/사주 그룹 리그

## 일일 루프 설계

### 아침 (7-9시)
1. 오늘의 운세 & 에너지 레벨 확인
2. 오늘의 3가지 추천 액션 확인
3. 캐릭터 인사 + 응원 메시지
4. 스트릭 체크인

### 낮 (12-14시) - 선택적
1. 중간 체크인 알림
2. "오전 어땠어요?" 간단 체크
3. 오후 에너지 예측

### 저녁 (21-23시)
1. 오늘 완료한 것 체크
2. XP 획득 & 레벨업 확인
3. 스트릭 업데이트
4. 내일 운세 미리보기
5. 캐릭터 굿나잇 메시지

## Duolingo 성공 공식 적용

```
손실 회피 (스트릭 끊김)
       +
사회적 비교 (리그/친구)
       +
즉각적 보상 (XP/레벨업)
       +
감정적 연결 (캐릭터 코칭)
       +
희소성 (하트/에너지 제한)
       ═══════════════
       극강의 리텐션
```

### 핵심 심리 요소

| 요소 | 구현 | 효과 |
|------|------|------|
| 손실 회피 | 스트릭 시스템 | 매일 접속 유도 |
| 가변 보상 | 가챠/뽑기 | 도파민 분비 |
| 사회적 증명 | 리그/순위 | 경쟁 욕구 |
| 소유 효과 | 캐릭터 육성 | 이탈 방지 |
| 목표 기울기 | 진행바/레벨 | 완료 욕구 |
| FOMO | 기간한정 이벤트 | 긴급성 |
