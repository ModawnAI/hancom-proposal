## Part 2: 실시간 Google Grounding 초개인화

### 2-1. Google Grounding이란?

#### 📖 개념 설명

**Grounding**은 AI 모델의 응답을 검증 가능한 정보 소스에 연결하는 기능입니다.

```
일반 AI 응답:
"오늘 날씨가 좋으니 외출하세요"
→ 실제 날씨 모름, 할루시네이션 가능

Grounding 적용 응답:
"오늘 서울 날씨가 맑고 최고 기온 15°C입니다 (출처: Weather.com).
외출하기 좋은 날씨네요."
→ 실시간 검색 기반, 출처 제공
```

#### 🔧 Vertex AI Grounding 종류

| Grounding 유형 | 설명 | 운세앱 활용도 |
|---------------|------|-------------|
| **Google Search** | 웹 검색 결과 연동 | ⭐⭐⭐⭐⭐ |
| **Google Maps** | 지리/위치 정보 | ⭐⭐⭐⭐ |
| **Vertex AI Search** | 자체 문서 검색 | ⭐⭐⭐ |
| **RAG Engine** | 커스텀 데이터 검색 | ⭐⭐⭐⭐ |

##### 예시 1: 날씨 + 사주 결합

- 위치: 서울 강남구

에너지 밸런스에 도움이 됩니다.

[출처: Weather.com - 서울 날씨]"
```

const { GoogleGenAI } = require('@google/genai');

const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY });

async function getGroundedFortune(userProfile, sajuAnalysis) {
  const prompt = `
    당신은 전문 운세 상담사입니다.
    
    사용자 사주: ${JSON.stringify(sajuAnalysis)}
    직업: ${userProfile.job}
    관심사: ${userProfile.interests.join(', ')}
    
    오늘의 운세를 실시간 정보와 결합하여 제공해주세요.
  `;

  const response = await ai.models.generateContent({
    model: 'gemini-2.5-flash',
    contents: [{ role: 'user', parts: [{ text: prompt }] }],
    tools: [{ googleSearch: {} }],
    toolConfig: {
      retrievalConfig: {
        latLng: {
          latitude: userProfile.location.lat,
          longitude: userProfile.location.lng
        }
      }
    }
  });

  return {
    fortune: response.candidates[0].content.parts[0].text,
    sources: response.candidates[0].groundingMetadata?.groundingChunks || []
  };
}
```

### B. 비용 계산기

```python
def estimate_monthly_cost(dau, queries_per_user, grounding_rate=0.5):
    """
    월간 비용 추정
    
    Args:
        dau: 일일 활성 사용자
        queries_per_user: 사용자당 일일 운세 요청 수
        grounding_rate: Grounding 사용 비율 (0-1)
    """
    daily_queries = dau * queries_per_user
    grounding_queries = daily_queries * grounding_rate
    monthly_grounding = grounding_queries * 30
    
    # Grounding 비용: $35/1K queries
    grounding_cost = (monthly_grounding / 1000) * 35
    
    # API 토큰 비용 (평균 1000 토큰/요청)
    monthly_tokens = daily_queries * 30 * 1000
    # Flash: $0.075/1M input, $0.30/1M output
    token_cost = (monthly_tokens / 1_000_000) * 0.375
    
    total = grounding_cost + token_cost
    
    return {
        "grounding_cost": grounding_cost,
        "token_cost": token_cost,
        "total_usd": total,
        "total_krw": total * 1300
    }

# 예시: DAU 10,000, 하루 3회 요청, 50% Grounding
result = estimate_monthly_cost(10000, 3, 0.5)
print(f"월간 예상 비용: ${result['total_usd']:,.0f} (₩{result['total_krw']:,.0f})")
```
