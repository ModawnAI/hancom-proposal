# HTML 제안서 통합 전략

## 1. 기존 파일 분석

### 현재 구조 (30 슬라이드)
| # | 유형 | 내용 |
|---|------|------|
| 0 | Cover | 표지 |
| 1 | 목차 | 분석 구성 |
| 2-3 | Executive Summary | 시장 핵심 지표 |
| 4-6 | 점신 | Section + 핵심지표 + SWOT |
| 7-10 | 포스텔러 | Section + 핵심지표 + FAS + SWOT |
| 11-13 | 청월당 | Section + 핵심지표 + SWOT |
| 14-17 | 헬로봇 | Section + 핵심지표 + SWOT + vs포스텔러 |
| 18-20 | 빅테크 | Section + 카카오/네이버 + 전문가 플랫폼 |
| 21-25 | 핵심 비교 | Section + 서비스/재무/기능/포지셔닝 |
| 26-29 | 신규 진입 | Section + 차별화 + 비즈니스모델 + 결론 |

### 기존 문제점
1. ❌ **전문가 상담을 장/단점으로 비교** → 우리는 AI 전용이므로 무의미
2. ❌ **해외 사례 부재** → Co-Star, Replika 등 없음
3. ❌ **AI 기술 관점 부족** → LLM, 데이터 수집 분석 없음
4. ❌ **우리 서비스 비전 없음** → 인생설계, 메타버스, Grounding 없음
5. ❌ **게이미피케이션 분석 부족** → 헬로봇만 언급
6. ❌ **SWOT이 1차원적** → AI 관점 재분석 필요

## 3. 핵심 수정 방향

### A. SWOT 재구성 (AI 관점)

**기존 (무의미한 항목):**
- ❌ "전문가 상담 400명+" → 우리는 AI 전용
- ❌ "오프라인 콜라보" → 우리 전략 아님

**새로운 기준:**
| 항목 | 분석 관점 |
|------|----------|
| AI 모델 | 규칙 기반 vs LLM vs 하이브리드 |
| 데이터 수집 | 수집 방식, 학습 루프, 개인화 수준 |
| 만세력 정밀도 | 시/분 단위, 천문학적 정확도 |
| 인생설계 연계 | 있음/없음 (대부분 없음) |
| 기술적 차별화 | 우리가 넘어설 수 있는 약점 |
