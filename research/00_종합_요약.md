# AI 인생설계 플랫폼 - 사업 제안서

> **작성일**: 2026년 2월  
> **목적**: 운세/점성술 시장의 검증된 성공 요소를 벤치마킹하고, 차별화된 기술력을 더해 새로운 가치를 창출하는 서비스 제안

---

## Executive Summary

### 시장 기회
국내 운세/사주 시장은 **4조원 이상** 규모의 검증된 시장입니다. 점신(1,900만 사용자), 포스텔러(900만), 헬로봇(600만) 등 주요 앱들이 각자의 강점으로 시장을 형성하고 있으며, 최근 청월당은 ChatGPT 스토어 글로벌 1위를 달성하며 AI 기반 운세의 가능성을 입증했습니다.

### 우리의 접근
15개 앱(국내 4개 + 해외 11개)의 성공 요소를 심층 분석하여, 각 앱이 사용자에게 사랑받는 이유를 파악했습니다. 이를 바탕으로 **검증된 강점들을 조합**하고, **우리만의 차별화 기술**을 더해 새로운 서비스를 제안합니다.

### 핵심 컨셉
**"당신의 사주가 알려주는 인생 설계"**

```
WHY (사주/운세)    +    WHAT (인생설계)    +    HOW (기술력)
      │                      │                     │
      ▼                      ▼                     ▼
  나에게 맞는            구체적 목표와          실시간 맥락화
  방향 발견              실행 계획              AI 기술
```

## 우리만의 차별화

### 1. 실시간 맥락화 운세 (Google Grounding)

**기존 운세앱:**
> "당신은 재성이 발달하여 금전운이 좋습니다."
> → 언제 봐도 같은 내용

**우리 서비스:**
> "오늘 서울 날씨가 맑고, 당신의 화(火) 기운과 조화롭습니다. 점심에 야외 미팅을 추천드려요. 참고로 오늘 코스피가 2% 상승 중인데, 재성이 강한 당신에게 투자 검토하기 좋은 타이밍입니다."
> → 실시간 정보와 연동된 "살아있는 운세"

### 2. 인생 설계 통합

```
인생 로드맵 (대운 기반)
    └── 10년 설계 (대운 시즌별 방향)
        └── 1년 설계 (세운 목표)
            └── 1달 설계 (월운 계획)
                └── 하루 설계 (오늘의 액션)
```

**점신의 깊이** + **Fabulous의 목표 시스템** + **대운/세운 타이밍** = 운세 기반 인생 계획

### 3. 메타버스 기술 적용

현재 메타버스 업계에서 검증된 기술들을 운세 도메인에 적용:

| 기술 | 적용 방식 |
|-----|----------|
| 디지털 DNA 아바타 | 사주 명식 기반 아바타 자동 생성 |
| AI 성격 반영 | 격국별 캐릭터 행동 패턴 |
| 시뮬레이터 | 대인관계, 직업, 연애 체험 |
| 대운 연동 성장 | 아바타가 대운 흐름에 따라 변화 |

**핵심 컨셉: "사주를 읽는 것 → 사주를 사는 것"**

### 4. 게이미피케이션 완전 통합

- **헬로봇의 캐릭터** + **Duolingo의 스트릭** + **운세 기반 퀘스트**
- 오행 밸런스 게이지, 일일 미션, 성장 시스템
- 매일 돌아오게 만드는 구조
