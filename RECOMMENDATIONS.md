# DITTO 제안서 개선 권고사항

## 현재 웹앱 vs 리서치 파일 비교 분석

---

## 1. 누락된 핵심 콘텐츠

### 1.1 킬러 피쳐 상세 설명 부족
**리서치에 있지만 웹앱에 없는 내용:**

| 기능 | 리서치 내용 | 현재 웹앱 상태 |
|------|-----------|--------------|
| 실시간 그라운딩 예시 | "금 기운 상승일 + 코스피 상승 = 투자 타이밍" 등 구체적 시나리오 | 개념만 언급 |
| AI 코치 캐릭터 | 오행 기반 5가지 코치 성격 상세 설명 | 미언급 |
| 게이미피케이션 상세 | 스트릭, 리그, XP 시스템, 레벨업 보상 | 간단히 언급만 |
| 사용자 시나리오 | 구체적인 하루 사용 흐름 | 미포함 |

### 1.2 차별화 포인트 강화 필요
**리서치의 핵심 메시지:**
- "운세를 확인하는 것" → "운세대로 행동하는 것"
- 사주가 WHY, 인생설계가 WHAT, AI가 HOW

**권고:** 솔루션 섹션에 이 프레임워크를 시각적으로 표현

## 2. 추가 권장 섹션

### 2.1 사용자 시나리오 섹션 (신규)
**리서치 파일:** `10_사용자_시나리오.md`

**포함할 내용:**
```
아침 7시: 오늘의 운세 푸시 알림
- "오늘 직장운 상승, 오후 2시 회의에서 적극적으로 발언해보세요"
- 오늘의 퀘스트: "동료에게 먼저 인사하기" (+15 XP)

점심 12시: AI 코치와 대화
- "점심 뭐 먹을까?" → "오늘 토 기운 부족, 노란색 음식 추천"

저녁 9시: 하루 리뷰
- 퀘스트 완료 체크, 스트릭 유지, 내일 운세 미리보기
```

### 2.2 게이미피케이션 상세 섹션 (확장)
**리서치 파일:** `09_게이미피케이션_설계.md`

**포함할 내용:**
- 스트릭 시스템 (7일/30일/100일/365일 보상)
- 인생 RPG 스탯 (직장운/건강운/재물운/연애운/성장운 = 오행)
- 12지신 캐릭터 수집 시스템
- 리그 시스템 (브론즈→실버→골드→다이아)

### 2.3 AI 코치 캐릭터 소개 (신규)
**리서치 파일:** `01_우리_서비스_비전.md`

**5가지 오행 코치:**
| 코치 | 오행 | 성격 | 전문 영역 |
|------|-----|-----|---------|
| 목(木) 코치 | 나무 | 성장 지향, 창의적 | 커리어, 학습 |
| 화(火) 코치 | 불 | 열정적, 활동적 | 연애, 사회관계 |
| 토(土) 코치 | 흙 | 안정적, 신뢰감 | 건강, 습관 |
| 금(金) 코치 | 금속 | 날카로운, 분석적 | 재테크, 결정 |
| 수(水) 코치 | 물 | 지혜로운, 유연한 | 정서, 명상 |

## 3. 기존 섹션 개선 제안

### 3.1 국내 앱 분석 섹션
**현재:** 4개 앱의 강점/약점/인사이트
**개선:** 각 앱에서 "우리가 배울 것" 명시

| 앱 | 우리가 가져올 것 |
|----|---------------|
| 점신 | 정확도와 깊이에 대한 신뢰 |
| 포스텔러 | 스토리텔링 전달 방식 |
| 청월당 | LLM 기술력 + 웹툰 시각화 |
| 헬로봇 | 캐릭터 친밀감 + 솔직한 조언 |

### 3.2 해외 앱 분석 섹션
**현재:** 4개 앱만 포함
**개선:** 카테고리별 구분 강화

**점성술 앱:** Co-Star, The Pattern
- 배울 점: 소셜 바이럴, 타이밍 사이클 개념

**AI 컴패니언:** Replika, Pi
- 배울 점: 메모리 기반 관계 형성, 24/7 가용성

**인생설계 앱:** Fabulous, Coach.me
- 배울 점: 행동과학 기반 습관 형성, 목표 추적

### 3.3 경쟁 공백 섹션
**현재:** 기능 매트릭스 표
**개선:** 블루오션 기회 강조

**핵심 메시지:**
> "전 세계 어떤 서비스도 운세 + 인생설계 + 게이미피케이션 + 실시간 맥락화를 결합하지 않았다"

시각화 제안: 4개 원이 겹치는 벤다이어그램 중앙에 "DITTO"

### 3.4 로드맵 섹션
**현재:** 2월→3월→4/1→5월+ 타임라인
**개선:** 각 단계별 구체적 산출물 명시

| 단계 | 기간 | 핵심 산출물 |
|-----|-----|-----------|
| MVP 개발 | 2월 | 만세력 엔진, AI 코치 프로토타입, 그라운딩 연동 |
| 베타 테스트 | 3월 | 내부 QA, 사주 정확도 검증, UX 피드백 |
| **제품 출시** | **4/1** | iOS/Android 앱, 웹 버전, 기본 게이미피케이션 |
| 기능 확장 | 5월~ | 소셜 기능, 궁합 매칭, AR 체험 |

## 4. 시각적 개선 제안

### 4.1 추가 SVG 다이어그램

**1) WHY-WHAT-HOW 프레임워크**
```
[사주/운세] ─WHY─▶ [인생설계] ─WHAT─▶ [AI 기술] ─HOW─▶ [행동]
```

**2) 하루 사용 흐름도**
```
아침 ──▶ 점심 ──▶ 저녁 ──▶ 밤
운세확인   AI대화   퀘스트    리뷰
```

**3) 게이미피케이션 피라미드**
```
        [레전드 보상]
       /            \
    [월간 목표]    [이벤트]
   /                      \
[주간 퀘스트]        [리그]
        |
   [일일 스트릭]
```

### 4.2 스크린샷/목업 추가
- 앱 UI 목업 (홈 화면, AI 대화, 운세 리포트)
- 캐릭터 디자인 컨셉
- AR 체험 예시 이미지

## 5. 콘텐츠 톤 & 메시지 개선

**현재:** "운세를 읽는 것에서 사는 것으로"
**유지:** 좋은 슬로건, 그대로 유지

- "매일 아침, 오늘의 최적 행동을 알려드립니다"
- "운세 + AI + 게임 = DITTO"

### 5.3 숫자로 말하기
리서치에서 가져올 수 있는 인상적인 수치:
- 국내 운세 시장: 4조원+
- 점신 다운로드: 1,900만+
- 헬로봇 평점: 4.7점 (12.4만 리뷰)
- 청월당: GPT 스토어 글로벌 1위

## 8. 결론

현재 웹앱은 **구조와 디자인이 잘 갖춰져 있으나**, 리서치에 담긴 풍부한 콘텐츠가 충분히 활용되지 않았습니다.

**가장 큰 개선 기회:**
1. **구체성 부족** - "게이미피케이션"이라고만 하지 말고 스트릭/리그/XP 등 상세 설명
2. **시나리오 부재** - 사용자가 실제로 어떻게 쓰는지 보여주기
3. **캐릭터 소개** - AI 코치 5종의 개성과 역할 설명
4. **블루오션 강조** - "아무도 하지 않은 조합"임을 시각적으로 강조

이 개선사항들을 적용하면 제안서의 설득력이 크게 향상될 것입니다.
